# QSVM — Network Intrusion Detection · Landing Page

Landing page for the Quantum SVM anomaly detection project.

**Live:** [qsvm-landing.vercel.app](https://qsvm-landing.vercel.app)  
**Model app:** [qsvm-anomaly-detection.streamlit.app](https://qsvm-anomaly-detection.streamlit.app)  
**Model repo:** [github.com/Bhargav-Sarvepalli/qsvm-anomaly-detection](https://github.com/Bhargav-Sarvepalli/qsvm-anomaly-detection)

---

## Stack

- React 18 + TypeScript + Vite
- Tailwind CSS
- Three.js + @react-three/fiber + @react-three/drei
- Framer Motion
- Lenis smooth scroll
- Vercel

## Run locally

```bash
npm install
npm run dev
```

## Deploy

Push to GitHub, connect to Vercel. Automatic deploys on every push to `main`.
